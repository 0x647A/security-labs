# security-labs

A collection of cybersecurity learning materials, scripts, and notes documenting my journey into security.

## Projects

### [sekurak](sekurak/)
Scripts and exercises from Sekurak courses and CTF challenges. Covers file system tools, web crawlers, password cracking experiments, multiprocessing demos, and FastAPI utilities. Built with Python.

### [CWL-notes](CWL-notes/)
Personal study notes from the **Cyber Women Leaders** program (Mamo Pracuj Foundation) — a free cybersecurity development program for women in Poland. Materials support CompTIA Security+ exam preparation, with lecture notes, glossaries, commands, and exercises in technical English.
