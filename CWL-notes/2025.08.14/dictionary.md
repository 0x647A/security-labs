# Glossary – Endpoint Security

## A
**AIDE** (Advanced Intrusion Detection Environment) – a Linux/Unix tool for monitoring system file integrity.

**AMSI** (Antimalware Scan Interface) – a Microsoft interface allowing real-time scanning of scripts (PowerShell, JavaScript, VBA) by antivirus solutions.

**AppArmor** – Linux access control system (primarily Ubuntu/Debian) that confines program permissions via security profiles.

**ASR** (Attack Surface Reduction) – Windows Defender features that reduce attack surface by blocking suspicious behaviors.

**ADS** (Alternate Data Streams) – an NTFS feature that allows hiding additional data streams within files.

## B
**BitLocker** – Microsoft disk encryption technology for Windows.

**BYOD** (Bring Your Own Device) – a policy permitting employees to use personal devices for work purposes.

## C
**C2** (Command and Control) – servers controlled by attackers to manage infected systems.

**CASB** (Cloud Access Security Broker) – solutions that secure access to cloud applications.

**CIS Benchmarks** – security configuration standards for various systems and applications published by the Center for Internet Security.

**Controlled Folder Access** – a Windows Defender feature protecting selected folders from ransomware.

**CVE** (Common Vulnerabilities and Exposures) – a public database of known security vulnerabilities.

**CVSS** (Common Vulnerability Scoring System) – a framework for rating the severity of vulnerabilities (scale 0–10).

## D
**DLP** (Data Loss Prevention) – technologies that prevent unauthorized transmission of sensitive data.

**docker.sock** – Unix socket for Docker daemon communication, often abused for privilege escalation attacks.

## E
**EDR** (Endpoint Detection & Response) – advanced endpoint security solutions with threat detection, analysis, and response capabilities.

**ETW** (Event Tracing for Windows) – a Windows framework for collecting and analyzing system events.

## F
**FileVault** – macOS full-disk encryption.

**Follina** – nickname for CVE-2022-30190, an Office vulnerability enabling remote code execution.

## G
**GPO** (Group Policy Object) – a Windows Active Directory mechanism for centralized configuration management.

## I
**Infostealer** – malware that steals user data (passwords, cookies, session tokens).

## J
**JA3** – TLS/SSL fingerprinting technique that identifies applications based on handshake characteristics.

## L
**LUKS** (Linux Unified Key Setup) – a standard for disk encryption on Linux.

## M
**MDM** (Mobile Device Management) – systems for managing mobile devices within organizations.

**MFA** (Multi-Factor Authentication) – authentication requiring multiple verification factors.

**Minifilter** – a Windows file system filter architecture used by antivirus products.

## N
**NDR** (Network Detection & Response) – solutions for detecting threats in network traffic.

**NLA** (Network Level Authentication) – an extra authentication layer for RDP connections.

## P
**Principle of Least Privilege** – security principle granting only minimal permissions required to perform tasks.

## R
**RaaS** (Ransomware-as-a-Service) – a business model allowing criminals to rent ransomware tools.

**Rootkit** – software tools that hide the presence of malware in a system.

**RWX** – memory/file permissions for Read, Write, and eXecute operations.

## S
**SCCM** (System Center Configuration Manager) – Microsoft platform for IT infrastructure management.

**seccomp** (secure computing) – a Linux kernel feature limiting available system calls for processes.

**SELinux** (Security-Enhanced Linux) – a Linux mandatory access control system.

**SIEM** (Security Information and Event Management) – systems for collecting, correlating, and analyzing security logs and events.

**SMBv1** – legacy version of the Server Message Block protocol vulnerable to various attacks.

**SRP** (Software Restriction Policies) – Windows mechanism restricting execution of unauthorized software.

**Sysmon** – a Microsoft tool for advanced system event logging.

## V
**VBA Stomping** – technique for hiding malicious VBA code by corrupting source code in Office documents.

## W
**WPAD** (Web Proxy Autodiscovery Protocol) – protocol for automatic proxy configuration often abused in attacks.

**WSUS** (Windows Server Update Services) – Microsoft service for centralized Windows update management.

## X
**XDR** (Extended Detection & Response) – solutions correlating data from endpoints, networks, cloud, email, and more to detect threats.

## Y
**YARA** – an open-source rule-based language and tool for identifying and classifying malware.

## Z
**Zero Trust** – a security model that assumes no implicit trust and verifies every request to access resources.
